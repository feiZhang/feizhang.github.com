2011-04-19

Asterisk部署终于基本完成了

总结下：

最大的问题： 1.客户端注册不稳定 ，第一次注册都成功，之后就逐渐掉线，直至全部掉线，我们有120个分机。 解决：是大唐高鸿的驻地网关的问题，软件版本2011-3-1的，有问题，降级解决，这个问题把我搞死了。

2.*8分组代答的问题。 刚开始，为了分组看起来有意义，就使用了字母作为分组：比如：YeWuZu1，不能代答，然后改为6680，跟坐席组号相同，也不行，最终改为1，可以了，真是奇怪呀，这个组又没啥用，为啥要这样弄呢？ 好奇怪的测试结果。。22可以、220 222 88 99都不可以，，33可以。y1不可以。

3.外呼，原来外呼只能一个，不能多路，运行商确认是外呼的时候，主叫号码设置有问题 之后使用 exten => _NXXXXXXX,1,Macro(user-callerid,) exten => _NXXXXXXX,n,Set(CALLERID(all)="6931nnnn" <6931nnnn>)

强制设定，可以了，很奇怪得是按说前面的6931设置后，尖括号内没必要用真实号码了，不清楚为啥驻地网关非要使用后面的号码为显示号码，应该还是大唐的问题，毕竟sip协议写的，前面是显示号码的。

4.拦截呼叫中心号码 这个简单，直接 exten => _6931nnnn,1,Goto(ext-queues,06800,1) 转到Queue中去就行了。

5.录音分目录存放，使用agi实现自动创建目录，返回路径即可

6.主叫号码，从中继(使用SIP账号登录)过来的呼叫，这边显示的主叫是系统设置的显示主叫，在Queue中更是显示的两个号码相加的号码，可以使用SIP_HEADER(From)获取，SIP的显示号码，通过截取获得真实主叫。

7.驻地网关中的拨号计划，不要短号码覆盖长号码，否则无法呼出长号码。

8.通过网关接入的电话，DTMF获取问题。在驻地网关，不一定都是要选择透传，这次选择rfc2833，估计是要和asterisk的配置一样。

9.A8里面需要设置代理服务器才能呼出，很奇怪的选项，如果没有，则使用注册服务器相同的地址。

10.ringgroup只能获取到1个呼入，跟下面所包含的分机数量无关，再来电话，则直接转到fail。

11.Queue只能在timeout时才能转到fail中去。。很失败。按道理，应该有轮询一遍或者全忙之后失败的选择。

12.T4使用SIP账号注册时，要设置呼出路由，才能呼出。

13.T4有访问控制功能，如果打开，则需要将sip服务器IP加入，否则无法注册成功。

14.单通的问题可能是中继线插错了

15.R32的拨号计划，先匹配短的，如果设置了：01XXX则，所有的长途号码都会被截断，比如：018633334444被截断为01863。拨号计划不分前后行，放在哪里都会被截断。

16.最近将Asterisk升级到1.8，R32无法注册，但是T4缺可以，要求大唐提供新软件，刷机后，可以了

17.astdb真是个麻烦的东西，备份的sip分机配置，无法自动恢复到astdb中，要么备份astdb文件，要么使用 /var/www/html/admin/modules/backup/bin/ 下面的备份和恢复脚本。既然有配置有astdb，asterisk就不能自动根据配置，刷新astdb？有点崩溃，没找到方法。

18.经常出现某个分机的通道挂起的状态，能外呼叫，不能接听。使用channel request hangup SIP/xxxxxxx 之后就行了，还没搞清楚为啥

还有一些其他基本功能，就相对简单点了。

呵呵，终于搞定asterisk的第一个应用了。真是一个强大的系统。。牛逼。。就是源代码太多了，无从下手，有空还是需要学习学习整体框架设计和具体的编码，假如能够看懂，那么内力应该会提升一大段。

2011-04-13

懒

虽然不是很喜欢看电视剧，但是还是对“很多剧情是误解推动的”挺有感触，很多的揪心剧情都是由各式各样的误解组合起来，让人无限的遗憾，恨不得去敲主角的脑袋，“咋就那么笨呢”，现实没有那么多交错的误解，不过还是不免有一些的。

但凡遇到这种情况，在已知的情况下，都会去解释的，可我不，不清楚为什么，人的信条？性格？懒惰？不在意？说不准。不过比较明确的是，信条：一般对人，我是采用总体评价的，如果对方可以获得我的信任，那么很ok，他该怎么做就怎么做去吧，一切我都能理解，直到不再信任，同时我认为人这种动物，要想在一生中有所重大的变化，相对来说是巨难的，所以对个人的态度，几乎是不用改变的。再则就是感觉，人生本来就是多样性，谁知道向左走的风景就一定比向右走的风景好多少呢？我就是我，让别人难过，不是我的个性，甚至我宁愿让自己稍微痛苦一点，不是高尚，只是本性，就如我现在不认为所谓的母爱父爱有多高尚一样，你不让他爱他难受，本质来自于本性和礼教，这样说看起来很悲剧，可是谁又能保证不是，谁又能说清楚呢？而过多的依赖自己的本性，而不去做相应的调整，比如溺爱等等，归根结底偶认为是懒，这样看来动物貌似是植物进化过来的，不是为了得到一些什么，都懒得动了。

这次，我就勤奋一次，解释一下“当时，我在搽桌子，刚搽完（1分钟不到），有人已经暴怒了，做相应的处理了，作罢”，挺无聊的吧。

在我如行尸走肉的这段日子里，其实干什么对我来说都是无所谓地。

2011-04-07

儿子的智商

小丁丁2岁多了，能说一些简单的话了，也就可以和其他人有些简单的交流了。初次感到丁丁的智商是我和他在外面玩耍的时候，当时只有我们两个，不记得是为了什么，他又一次任性的趴到了地上，无论如何就不起来，偶怒了，屁股上给了一巴掌（不重），命令他“起来”，他居然真的就爬起来了（以前他才不鸟你呢），再接再厉，命令道“不许哭”，结果人家就硬生生的停止了哭，抱着他走的时候，我忍不住笑了一路，这家伙还是很好对付的。而在家里不好对付的原因，应该就是有其他人在的缘故，他会期待其他人护着他。教育这个大难题，应该难就难在不能一以贯之。

第二次同样的情况，同样的命令，依然好使，前提是必须只有我们两个，看来这家伙真的长大了，有智商了，而所谓的教育也应该开始了。而应该从哪里开始，要培养什么却不是我所知道的，而像我现在的意识形态，是倾向于“不教”的，所有的教育来自于他的眼睛，来自于他的思考。也许我们可以在处理一些事情的时候在一起探讨而已。事实却肯定不能如愿，周围的其他人都还是倾向于子不教父之过的，而且都很有好坏之分，让我感觉自己很不是人，反正我给自己的任务就是：能尽量的增加和他在一起玩耍的时间，能让他吃饱、不冷就行，而其他的，我就爱莫能助了。

貌似现在自己还是有点容易怒，虽然已经改观很对了，其实想来，这样的改观，有一部分来自于自己的需要，而更大的原因应该是我对于现实的无奈，无论怎么做，其实结果又能改变多少，还不如放弃来的划算。想放弃，而又不能放弃，无奈无聊无助，就是不能无法无天，现在想来，想做个混蛋真的很难，貌似什么都是注定的。

2011-03-24

三十不惑

看起来有些斗胆了。
最近很少更新博客，其实有感触的事挺多的，可是，一开始写就会有“没必要”的感觉。现在对于很多事情，稍想，就都不太在意了。
总是感觉，事情的轮回根本就不是人能确定的，我们能做的仅仅就是随着时间流淌，“快乐的猪和痛苦的思考者”，其实无论你羡慕那个，都无所谓了，当你想起这个问题的时候，你已经是一个痛苦的思考者了，而当你不再思考类似问题的时候，你也许就是一只快乐的猪了，所以，当下的想法总是与现实有那么些差距。而又无需改变，变化后的结果无非有两种，“想法总是与现实有那么些差距”，“放弃想法”，既然放弃了想法，那么你就可以成为一只披着人皮的羊，不能和“人”同比了。
谁知道呢，反正现在的状态还挺不错的，也许是因为生活条件改善了，没有必要去忧虑太多的事情，所以能够稍微宁静些，考虑那么多干甚，现实已经摆在眼前。
不惑归不惑吧，装逼还得继续，遵从内心的呼唤，我要说：“你们这群傻逼”
这群傻逼在抢盐的风波中，将自己的风格展现的淋漓尽致，真他妈的傻逼的木法说了。拨开现象看本质，也还是“弱肉强势”这千古不变的现象，一部分所谓的精英，设计各种的圈套去宰杀傻逼的弱势群体，而这批弱势群体并不引以为戒，天天上套不说，还口口声声，要申请“主子来保护”，声称国家没有及时声明，我靠，你还不如说：“我只是一直快乐的猪，我啥都不知道，你把我养起来吧，等瘦肉长多了，可以杀杀的”，xx的，什么玩意。
写到这，我又不想写了，狗屁说这些有什么用呢？该怎么还是怎么样，都是浮云。而且看起来貌似我还挺愤青，球，爷只是图个自己爽爽。
推荐一下，网易新闻-有态度的新闻，可以看下这篇：http://news.163.com/special/reviews/oilwar.html 确认下你到底属于哪类人，偶尔也挺羡慕那些阴谋论者，他们大多都很有市场，在诸多场合下都会获得共鸣，而很多深刻的思考往往会被质疑甚至鄙视，太多人，不知道思考才是人与动物的区分。既然为人，就好好做人。
其实，说这么多都他妈是一个遮羞布，偶很阿Q，不管自己的事向来是不上心的，只是感觉周围不会独立思考的人是如此之多，让偶很有孤独感，所谓的兄弟们，还如10年以前，但是没有10年以前的纯真，但是却一直放不下所谓的“牛逼”，都他妈的30多的人了，还不明白，牛逼不是吹出来的，是真刀实枪干出来的，如果你真想“牛逼”，就他妈的好好干，否则，就仔细想想，该干些真正正常的事了。因为，这个世界不是你一个人在战斗，放眼望去，都是当爹当妈的人了，还在那蹉跎岁月，不是扯淡。其实，关键的关键，你真是那个牛逼的人吗？很简单，绝大多数的人都不是，也许你会获得释兄的偶尔垂青，那也仅仅是踩到了狗屎，每天都在重复着无聊事情，不提高自己的认知和常识，想牛逼，那不就是扯淡。
谁知道，我是不是在犯本位主义，谁知道，我他妈的为啥非要说这些，也许是，我看起来有些孤独了，我已经厌倦了，谁谁谁牛逼这套东西。兄弟们，看清自己，我们只是个巨普通的人。废话了这么多，我就想说“你们这群傻逼，让我们在一起快乐一点，不要再装牛逼”。

转 四十不惑

一个前辈写的，有感触.

“四十不惑 。不惑，就是明白，说人到了四十岁对什么事都明白了，对自己看得更清了，就大彻大悟了，把什么都看得平淡了，波澜不变，荣辱不惊，一切泰然处之。一位哲人对一个四十岁的说过这么一句话：四十岁正是一个以不变应万变的年龄，四十岁是一个可以洞察世间万物的年龄，四十岁是一个成熟的年龄。成熟往往意味着冷漠、意味着明哲保身、意味着前怕狼后怕虎。平时怒见不平一声吼的人大多是那些初出茅庐的热血青年，铮铮直言的人恰恰是那些垂暮老者。而四十岁的人有着牛的忍辱，更多的是牛的懦弱，你啥时看到过当一只猛虎来犯的时候，冲上去的是一群牛？。这说明四十岁的男人身上多了稳重，少了冲动，多了成熟，而少了血性。与三十岁的男人不同的是他们春风得意但不忘乎所以，他们会把一件事正过来看反过来看，透过现象看本质，会看清自己，懂得如何把自己的那一面隐藏起来，而把另外一面示于人。"
