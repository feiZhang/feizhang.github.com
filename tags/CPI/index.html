物价高涨,为什么?

1.大多数人认为是货币超发，通货膨胀
那么我想问，这些货币发给谁了，理理。
4万亿投资 -> 工程商、民工、原产料生产企业获利 -> 这些人的购买力提升 -> 推动物价上涨，那么这个结论应该是说，国家超印了很多钱，将这部分钱用在了公共建设方面，而与此相关的人员通过劳动获取了这部分货币，提高了自己的购买力，从而推动物价上涨，而与此投资无关的人员，则被动接受通胀，导致购买力下降，但是获得了使用更好的公共基础设施的权利，，个人感觉没啥问题呀，涨就涨白，过个3、5年，货币还是会流动到他该去的地方，不影响什么实质内容，国家激动什么呢？全民教育下不就得了。最终只不过是纸币的贬值，而不影响实际生活，感觉还好呀，至少出国旅游更便宜了。
2. 人为炒作
这个我感觉是主因，毕竟国人还是热衷于储蓄的，如果突然提高了购买力，应该不会突然长期改变消费习惯，原来一月吃一斤肉，突然改为吃五斤，感觉不太符合常理，我更相信，“某些人炒作了某些商品，而国家所谓的抑CPI的举措，更加深了人民涨价的恐惧，从而从众涨价 ”，这个就更好解决了，由市场来处理嘛，如果养猪是暴利行业，就会有更多人进入，从而供过于求，降价，多正常的流程，何必担忧？

搞不懂呀搞不懂，我还是相信高层的智商比我高很多地，自己随便瞎诌一下，望高人指点。
