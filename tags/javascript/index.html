一个Js的奇怪错误，在Unix系执行

最近项目总是出现一个js的奇怪错误，在公司的服务器部署，就没有问题，而放到公网，就会出现莫名其妙的错误，比如:break不行等等，按照ie提示的位置去看，代码是没有问题的，firefox下没有问题。

偶然间发现，可能是文件编码的问题，在我机器上eclipse打开，中文乱码，但是这些中文都是被注释掉的，随改变编码，复制，将文件修改为项目相同的编码gbk，粘贴，再部署，居然问题就没了。。

看来在unix+was下，js如果被认为编码不符，也会让ie执行不正常，即便是注释，这个问题很诧异。
